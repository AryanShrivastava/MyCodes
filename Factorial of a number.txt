Factorial
Easy (GFG)

Given a positive integer, N. Find the factorial of N.
 

Example 1:

Input:
N = 5
Output:
120
Explanation:
5*4*3*2*1 = 120

long long int factorial(int N){
       //code here
       long long int res = 1;
       for(int i =2 ; i<= N ;i++)
       {
           res = res *  i;
       }
       return res;
}
