Missing number in array
Easy (GfG)

Given an array of size N-1 such that it only contains distinct integers in the range of 1 to N. Find the missing element.

Example 1:

Input:
N = 5
A[] = {1,2,3,5}
Output: 4

int MissingNumber(vector<int>& array, int n) {
    // Your code goes here
    int sum = 0 , sums = 0;
    sum = n * (n + 1) / 2;
    for(int i=0 ; i< n-1 ; i++)
    {
        
        sums += array[i];
        
    }
    return abs(sum - sums);  
}
